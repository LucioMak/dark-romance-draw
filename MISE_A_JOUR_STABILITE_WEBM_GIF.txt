Correctif stabilité médias - Dark Romance Draw

Fichiers à remplacer sur GitHub :

- style.css
- service-worker.js

Ce correctif :

- lit les WEBM avec une vraie balise vidéo, en boucle, muette et compatible mobile ;

Après mise en ligne : attendre 1 à 3 minutes, ouvrir l'application dans Chrome, puis Actualiser deux fois.
Si l'ancienne version reste affichée, fermer Chrome puis rouvrir le lien GitHub Pages.
