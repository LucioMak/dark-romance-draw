DARK ROMANCE DRAW - VERSION CLE EN MAIN

OPTION LA PLUS SIMPLE POUR JOUER TOUT DE SUITE SUR PC :
1. Dezippe le fichier.
2. Ouvre le dossier.
3. Double-clique sur LANCER_WINDOWS.bat si tu es sur Windows.
4. L'application s'ouvre dans ton navigateur.
5. Au premier lancement, choisis ton code PIN a 4 chiffres.

SI TU ES SUR MAC OU LINUX :
Double-clique sur LANCER_MAC_LINUX.command.

SI TU VEUX L'AVOIR SUR ANDROID COMME UNE APPLICATION :
Le plus simple est de la mettre en ligne avec Netlify Drop.
1. Va sur https://app.netlify.com/drop
2. Glisse le dossier de l'application dedans, ou le zip.
3. Netlify te donne un lien.
4. Ouvre ce lien sur Chrome Android.
5. Menu ⋮ > Ajouter a l'ecran d'accueil / Installer l'application.

SI TU VEUX GITHUB :
1. Va sur github.com
2. Cree un nouveau repository.
3. Upload tous les fichiers de ce dossier a la racine du repository.
4. Va dans Settings > Pages.
5. Source : Deploy from branch.
6. Branche : main, dossier : /root.
7. Attends quelques minutes, GitHub donne un lien public.

IMPORTANT :
- Tes GIFs et videos sont stockes localement dans le navigateur de l'appareil.
- Si tu ouvres l'application sur un autre telephone, il faudra reimporter tes medias ou utiliser l'export/import.
- Ne vide pas les donnees du navigateur si tu veux garder tes medias.

FORMATS :
GIF, WebP anime, MP4, WebM, et certains MOV/M4V selon le navigateur.

Le fichier important est index.html. Il doit rester a la racine.
